# Smart Tax Tools

Этот репозиторий содержит учебный проект по разработке и автоматизации Python-библиотеки для финансовых расчетов

## Структура репозитория
* **`smart_tax_tools/`** — Основная директория проекта.
    * `src/` — Исходный код библиотеки.
    * `tests/` — Модульные тесты (pytest).
    * `README.md` — Статья по проекту.
* **`Makefile`** — Файл автоматизации всех действий.
* **`.gitignore`** — Сгенерирован с помощью сайта, содержит все необходимое.

## Быстрый запуск
Для полной проверки проекта выполните из корня:
```bash
make install
make all
```

## Установка
Библиотека установлена на [TestPyPI](https://test.pypi.org/project/smart-tax-tools/0.0.3/), для установки используйте команду:
```bash
pip install --index-url https://test.pypi.org/simple/ smart-tax-tools==0.0.3
```
